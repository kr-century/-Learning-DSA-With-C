## 🚀 How to Deploy on GitHub Pages

### **Option 1: Using GitHub Web Interface (Easiest)**

1. **Go to your GitHub repository**
   - Create a new repository or use existing one

2. **Upload the HTML file**
   - Click "Add file" → "Upload files"
   - Drag and drop `DSA_C_Complete_Roadmap.html`
   - Commit the changes

3. **Enable GitHub Pages**
   - Go to Settings → Pages
   - Select "main" or "master" branch as source
   - Save and wait 2-3 minutes
   - Your site will be live at: `https://yourusername.github.io/repository-name/DSA_C_Complete_Roadmap.html`

### **Option 2: Using Git Command Line**

```bash
# Clone your repository
git clone https://github.com/yourusername/your-repo.git
cd your-repo

# Copy the HTML file
cp /path/to/DSA_C_Complete_Roadmap.html .

# Create index.html (optional - for direct access)
cp DSA_C_Complete_Roadmap.html index.html

# Add and commit
git add DSA_C_Complete_Roadmap.html
git commit -m "Add: Complete DSA in C Learning Roadmap"
git push origin main

# Or push to specific branch
git push origin gh-pages
```

### **Option 3: Rename to index.html (For Clean URL)**

```bash
# Rename the file to index.html for direct access
mv DSA_C_Complete_Roadmap.html index.html

# Now access at: https://yourusername.github.io/repository-name/
# Instead of: https://yourusername.github.io/repository-name/DSA_C_Complete_Roadmap.html
```

### **Option 4: Using GitHub Pages Direct Upload**

1. Create repository with name: `yourusername.github.io`
2. Upload `index.html` (renamed from DSA_C_Complete_Roadmap.html)
3. Access at: `https://yourusername.github.io`
